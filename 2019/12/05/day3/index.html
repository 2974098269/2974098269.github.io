IP地址介绍及VLSM

CVE _2019_07_08

CVE _2019_07_08

IP地址由32位二进制数组成，这些位被划分为4个字节，中间用“.”隔开，

分为网络地址和主机地址两个部分。IPV4一共有：2^32 大约是40亿多个地址

IP地址在网路是唯一的，不能重复。

对地址进行分类

公有地址、私有地址、保留地址

NET网络地址转换

IP地址分类

A类地址分类

第一个字节的第一位必须为0，第一个字节的取值为0-127

主机号全0是网络号（1.0.0.0） 全1是广播地址（1.1.1.1） 255.0.0.0 前八位是网络号 后24位是主机地址

在网络中有 2^24 个地址可用

有效地址为（可以配置给主机的地址）：2^24-2

子网掩码为：255.0.0.0 斜杠表示法：1.1.1.1/8

B类地址分类

第一个字节的第一位必须是1，且第二位必须是0，第一个字节的取值为128-191

主机号全0是网络号（1.1.0.0） 全1是广播地址（1.1.1.1） 255.0.0.0 前八位是网络号 后24位是主机地址

在网络中有 2^24 个地址可用

有效地址为（可以配置给主机的地址）：2^24-2

子网掩码为：255.0.0.0 斜杠表示法：1.1.1.1/8

c类地址分累

第一个字节的前前两位必须是1，且第三位必须是0，第一个字节的取值为192-223

D类网络地址范围：第一个字节为224-239，用作组播地址。

E类网络的地址范围：第一个字节为240-255，用于科学用途。

保留地址的IP

网络地址为0：表式当前网络或网段

网咯地址全为一：表示所有网络

地址127.0.0.1 保留用于环回测试

节点地址全为0：表示网络地址或指定网络中的任何主机

私有IP地址范围

A类 10.0.0.0~10.255.255.255

B类 172.16.0.0~172.31.255.255

C类 192.168.0.0~192.168.255.255

子网划分

将大的网络划分成若干个小网络，尽量减少广播风暴，提高网络效率

比如：

c类IP地址：192.168.1.0 网络号：192.168.1.0 广播号：192.168.1.255 可用地址：1.1-1.254

交换机工作原理及VLAN

以太网地址用来识别一个以太网上的某个单独的设备或一组设备

MAC地址是有48位二进制数组成，以十六进制数表示。钱24位称为OUI，后24位是厂商对网卡的唯一标识，第八位如果是0表示单播地址。

以太网帧格式

前导码（7字节）

交换机作用：分割冲突域，实现全双工通信，过滤转发。

先广播 知道mac地址后就单播 mac地址老化时间是300秒vlan

交换机的配置

交换机的所有端口默认情况下都在一个网络（vlan1），所以只要将电脑连接到交换机，并配置同一个网络内的地址，这样就形成了一个局域网。所以这样会形成广播风暴。

交换机的分类：傻瓜交换机

​ 可管理型交换机（二层）

​ 三层交换机：带路由功能的交换机

1.vlan技术（虚拟局域网）

如何实现不同交换机上的同一个**vlan**间的通信？

需要使用trunk【vlan中继协议】技术

（1**）封装**

ISL（思科私有）：是在数据帧前加26字节，尾加4字节

802.1Q（工业标准）：是在数据帧中插入4字节，打标签

802.1Q Native Vlan：默认vlan1不打标，作管理用

（2**）模式**

Mode DTP send DTP Response

-———————————————–

Access - -

Trunk Y Y

Desirable Y Y 企望

Auto N Y 自动

Nonegotiate N N 不协商
